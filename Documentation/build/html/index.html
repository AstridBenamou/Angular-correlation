lib.make_bins(data, r_max, dr)¶

Returns array of bins containing for each the number of points seperated by a distance between r and dr.

Parameters

  • data (numpy array) – array of shape = [n,3] containing coordinates of points in box

  • r_max (float) – distance max to search for points separation

  • dr (float) – size of bins (thickness of layers in which to search for points in each bins)

Returns

Bins chart with for each i the number of points separated by a distance between i*dr and (i+1)*dr

Return type

numpy array
